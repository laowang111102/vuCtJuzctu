# 知识图谱可视化习题推荐系统 python638

## 项目概述

知识图谱可视化习题推荐系统是一款集知识点可视化展示、智能问答、习题推荐于一体的教育辅助工具。它通过结合Python、Vue和Neo4j技术，为学习者提供了一个清晰直观、互动性强的学习平台。

## 技术栈

- 前端：Vue
- 后端：Python
- 数据库：Neo4j

## 功能模块

### 知识点可视化展示

- 以思维导图形式呈现知识点，帮助学习者构建知识框架。

### 智能问答

- 解答学习者在学习过程中遇到的具体问题，提供即时帮助。

### 习题推荐

- 根据学习者的学习水平和进度，个性化推荐习题，巩固知识点。

## 系统角色

- 学习者：使用系统的主体，可查看知识点、提问问题并完成习题。

## 运行环境

- 前端：现代浏览器
- 后端：Python环境（建议使用Python 3.x版本）
- 数据库：Neo4j数据库服务

## 部署步骤

1. 确保Neo4j数据库服务已正确安装并运行。
2. 搭建Python后端环境，安装必要的依赖。
3. 部署Vue前端项目。
